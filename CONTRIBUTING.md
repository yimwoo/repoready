# Contributing to RepoReady

Thanks for your interest in improving RepoReady.

## How to contribute
- open an issue for bugs or feature ideas
- suggest better repo-readiness checks
- improve packaging heuristics and suggestions
- help with examples and docs

## Development
```bash
python3 repoready.py .
```

## Goal
Keep RepoReady simple, useful, and focused on helping repos make a stronger first impression.
